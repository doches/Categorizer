Oracle

Abstracts the oracle data in data/categories.yaml. This class is mainly used for evaluation (when we want to compare the performance of a model against our human participants) or for training a supervised or semi-supervised model. As all of my models are currently at least semi-supervised, this gets used a lot...

You don’t instantiate an Oracle — instead, use the class methods to access whatever piece of data you need. The actual loading of data is handled internally by a singleton object, so multiple, independent references to oracle data only require a single loading of the file.

  def Oracle.exemplars  
    Oracle.init
    exemplars = []
    @@yaml.each_pair do |category,list|
      list.each_pair do |word,freq|
        exemplars.push Exemplar.new(word,category,freq)
      end
    end
    return exemplars
  end
