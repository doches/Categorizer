Models

Helper class to create a specific model from a symbolic identifier.

Public Class Methods

load(sym) click to toggle source

Load a model dynamically based on a symbol. Models.load(sym) will return an instance of the model specified by sym, as specified in the following scheme.

sym
A symbol of the form :__. The components of the symbol are transformed into a intercapped class name with “Model” appended, so :lsa_cluto_rb becomes LsaClutoRbModel 
# File lib/Models.rb, line 14
  def Models.load(sym)
    return eval("#{Models.sym_to_class(sym)}Model.new")
  end
output_path(sym) click to toggle source

Shortcut for producing a model-specific filename used for reporting.

sym
A symbol of the same form as used in Models.load.

For example, the symbol :lsa_cluto_rb will return the filename “LsaClutoRbModel.report“ 

# File lib/Models.rb, line 23
  def Models.output_path(sym)
    return Models.sym_to_class(sym)+".report"
  end
sym_to_class(sym) click to toggle source

Used internally to transform symbolic names (i.e. :lsa_cluto_rb) into intercapped model class names (i.e. LsaClutoRbModel). Returns a string of the class name. 

# File lib/Models.rb, line 28
  def Models.sym_to_class(sym)
    return sym.to_s.gsub(":","").split("_").map { |x| x.capitalize }.join("")
  end
